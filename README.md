
# Belajar MERN

Sebuah repo hasil belajar MERN (MySQL Express React Node) JS


## Run Locally

Clone the project

```bash
  git clone https://github.com/ibrahimrangkuti/mern-backend.git
```

Go to the project directory

```bash
  cd mern-backend
```

Install dependencies

```bash
  npm install
```

Install nodemon

```bash
  npm install -g nodemon
```

Start the server

```bash
  nodemon index
```
